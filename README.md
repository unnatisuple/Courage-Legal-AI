# ⚖️ Courage: Simplifying Justice for You

Courage is an AI-powered legal assistance platform designed to make legal information and support more accessible to citizens across India. The platform provides multilingual assistance in **Hindi, Marathi, Telugu, and English**, helping users understand legal processes, access government services, and connect with support organizations.

## 🌟 Features

* 🤖 AI-powered legal guidance
* 🌐 Multilingual support (English, Hindi, Marathi, Telugu)
* 📄 Assistance with FIR-related information
* 📝 RTI (Right to Information) guidance
* 🏛️ Information about legal rights and procedures
* 🤝 NGO support and resource discovery
* 📱 Responsive and user-friendly interface
* ⚡ Fast and accessible web application

---

## 🎯 Problem Statement

Many citizens face challenges in accessing legal information due to:

* Language barriers
* Complex legal terminology
* Lack of awareness about rights and procedures
* Difficulty finding reliable legal resources

Courage aims to bridge this gap by providing easy-to-understand legal assistance through AI and regional language support.

## 🚀 Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/yourusername/courage.git
cd courage
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
